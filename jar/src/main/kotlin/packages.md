# Package jline.api.cache

Cache modeling algorithms and performance analysis methods.

This package provides algorithms for analyzing cache systems and their
performance characteristics. It includes methods for evaluating hit rates,
miss penalties, and overall cache effectiveness in queueing network models.

# Package jline.api.lossn

Loss network analysis algorithms.

This package provides algorithms for analyzing loss networks where arrivals can
be blocked when resources are unavailable. Implements Erlang fixed-point methods.

# Package jline.api.lsn

Layered Stochastic Network (LSN) analysis algorithms.

This package provides algorithms for analyzing layered stochastic networks
and layered queueing networks (LQNs), including multiplicity analysis.

# Package jline.api.mam

Matrix Analytic Methods (MAM) for structured Markov chains.

This package implements matrix-analytic methods for solving structured
Markov chains, including quasi-birth-death (QBD) processes, MAP/MAP/1 queues,
and related models. These methods are efficient for analyzing systems with
repetitive state space structures.

# Package jline.api.mam.m3pp

M3PP (Marked Markov Modulated Poisson Process) fitting and analysis.

This package provides algorithms for fitting and analyzing M3PP processes,
which are extensions of MMPPs to marked arrival processes.

# Package jline.api.mapqn

MAP Queueing Network analysis algorithms.

This package provides algorithms for analyzing queueing networks with
Markovian Arrival Processes (MAP) and related process types.

# Package jline.api.mc

Markov Chain analysis utilities.

This package provides general tools for analyzing Markov chains, including
steady-state and transient analysis for both continuous-time (CTMC) and
discrete-time (DTMC) Markov chains.

# Package jline.api.measures

Performance measures and metrics computation.

This package provides utilities for computing and organizing performance
measures such as throughput, utilization, response time, and queue length.

# Package jline.api.npfqn

Non-Product-Form Queueing Network (NPFQN) approximations.

This package provides approximation algorithms for queueing networks
that do not satisfy product-form conditions, including traffic merging
and splitting operations for non-exponential service times.

# Package jline.api.pfqn

Product-Form Queueing Network (PFQN) analysis algorithms.

This package contains algorithms for analyzing queueing networks with
product-form solutions, including Mean Value Analysis (MVA), normalizing
constant, and convolution algorithms.

# Package jline.api.pfqn.lcfs

LCFS (Last-Come-First-Served) queueing network algorithms.

This package provides algorithms for analyzing closed queueing networks
with LCFS scheduling disciplines, including MVA and normalizing constant
computation methods.

# Package jline.api.pfqn.ld

Load-dependent queueing network algorithms.

This package provides algorithms for analyzing queueing networks with
load-dependent service rates, including extensions of MVA for variable
service capacities.

# Package jline.api.pfqn.mva

Mean Value Analysis (MVA) algorithms.

This package contains various MVA algorithms for analyzing closed queueing
networks, including exact MVA, approximate MVA, and extensions for
load-dependent and multi-server systems.

# Package jline.api.pfqn.nc

Normalizing Constant computation algorithms.

This package provides algorithms for computing normalizing constants of
product-form queueing networks, including convolution and recursive methods.

# Package jline.api.polling

Polling system analysis algorithms.

This package implements algorithms for analyzing polling systems with various
polling strategies including gated, exhaustive, and k-limited service disciplines.

# Package jline.api.qsys

Single queueing system analysis.

This package provides exact and approximate formulas for single queueing
systems such as M/M/c, M/G/1, G/G/1, and their variants.

# Package jline.api.rl

Reinforcement Learning utilities for queueing control.

This package provides utilities for applying reinforcement learning
techniques to queueing system control and optimization problems.

# Package jline.api.sn

Stochastic Network utility functions.

This package contains utility functions for stochastic networks, including
routing probability calculations, visit ratios, and network transformations.

# Package jline.api.trace

Trace data analysis functions.

This package provides functions for analyzing empirical trace data,
including statistical measures and transformations of time series
and event traces.

# Package jline.api.wf

Workflow modeling utilities.

This package provides utilities for modeling workflow systems and
their performance characteristics.

# Package jline.lib.butools

BuTools library port for Markovian modeling.

This package contains a port of the BuTools library, providing
algorithms for matrix-analytic methods and Markovian modeling.

# Package jline.lib.empht

EMpht algorithm for phase-type distribution fitting.

This package implements the EMpht algorithm for fitting phase-type
distributions to empirical data using expectation-maximization.

# Package jline.lib.lti

Linear Time-Invariant (LTI) system utilities.

This package provides utilities for analyzing linear time-invariant
systems, including matrix equations and Lyapunov solvers.

# Package jline.lib.m3a

M3A library for Markovian arrival process fitting.

This package provides algorithms for fitting Markovian arrival processes
to empirical data using moment matching and other techniques.

# Package jline.lib.perm

Matrix permanent computation algorithms.

This package provides efficient algorithms for computing matrix permanents,
used in various queueing network analysis methods.

# Package jline.lib.smc

SMC (Structured Markov Chain) solver library.

This package provides solvers for structured Markov chains, including
QBD processes and related models.
